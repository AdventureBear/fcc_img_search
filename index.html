FCC Flickr Image Abstraction Layer API

User stories
  • I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
  • I can paginate through the responses by adding a ?page=2 parameter to the URL
  • I can get a list of the most recently submitted search strings

To use:

Search Flickr tags and text for any search string:
https://fierce-lake-15290.herokuapp.com/api/imagesearch/aussiedoodle


Paginate through responses by sending the number of responses per page with "per_page"
https://fierce-lake-15290.herokuapp.com/api/imagesearch/dolphins?per_page=10


Paginate through responses by sending the page request with "page"
https://fierce-lake-15290.herokuapp.com/api/imagesearch/trout?page=2

See the search history:
https://fierce-lake-15290.herokuapp.com/api/latest
